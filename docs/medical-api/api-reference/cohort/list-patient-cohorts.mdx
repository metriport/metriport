---
title: "List Patient Cohorts"
description: "Returns a list of all cohorts the patient is a member of. For more information, see the [guide about Cohorts](/medical-api/handling-data/cohorts)"
api: "GET /medical/v1/patient/{id}/cohort"
---

## Path Params

<ParamField path="id" type="string" required>
  The ID of the patient whose cohorts are to be returned.
</ParamField>
