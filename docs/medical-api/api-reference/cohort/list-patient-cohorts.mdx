---
title: "List Patient Cohorts"
description: "Returns a list of all cohorts the patient is a member of."
api: "GET /medical/v1/patient/{id}/cohorts"
---

For more information, see the [guide about Cohorts](/medical-api/handling-data/cohorts)

## Path Params

<ParamField path="id" type="string" required>
  The ID of the patient whose cohorts are to be returned.
</ParamField>

## Response

<ResponseField name="cohorts" type="Cohort[]" required>
  Array of cohorts the patient is assigned to.

import { MetriportMedicalApi } from "@metriport/api-sdk";
