---
title: "Remove Patients from Cohort"
description: "Bulk remove patients from a cohort."
api: "DELETE /medical/v1/cohort/{id}/patient"
---

## Path Params

<ParamField path="id" type="string" required>
  The ID of the cohort to remove patients from.
</ParamField>

## Body

<ParamField body="patientIds" type="string[]" required>
  Array of patient IDs to remove from the cohort.
</ParamField>

## Response

<ResponseField name="message" type="string" required>
  Confirmation message indicating the operation was
  successful.
</ResponseField>

<ResponseField name="removedCount" type="number" required>
  The number of patients that were successfully removed from
  the cohort.
</ResponseField>
