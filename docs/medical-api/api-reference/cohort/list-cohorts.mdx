---
title: "List Cohorts"
description: "Lists all cohorts defined by your organization, or returns a specific cohort by name."
api: "GET /medical/v1/cohort"
---

## Query Parameters

<ParamField query="name" type="string" optional>
  The name of a specific cohort to return. If provided, returns only the cohort with the specified name instead of all cohorts.
</ParamField>

## Response

<ResponseField name="cohorts" type="Cohort[]">
  Array of cohorts with count of patients assigned to them.

</ResponseField>

<ResponseExample>

```javascript Metriport SDK
import { MetriportMedicalApi } from "@metriport/api-sdk";

const metriport = new MetriportMedicalApi("YOUR_API_KEY");

// List all cohorts
const allCohorts = await metriport.listCohorts();

// Get a specific cohort by name
const specificCohort = await metriport.getCohortByName({ name: "Pilot - Eval" });
```
