---
title: "List Patient Cohorts"
description: "Returns a list of all cohorts the patient is a member of. Click for additional information on [Cohorts](/medical-api/handling-data/cohorts)"
api: "GET /medical/v1/patient/{id}/cohort"
---

## Path Params

<ParamField path="id" type="string" required>
  The ID of the patient whose cohorts are to be returned.
</ParamField>

## Response

<ResponseField name="cohorts" type="Cohort[]" required>
