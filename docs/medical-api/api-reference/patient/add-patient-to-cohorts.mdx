---
title: "Add Patient to Cohorts"
description: "Adds a single patient to multiple cohorts at once. Returns the cohorts the patient was successfully added to."
api: "POST /medical/v1/patient/{id}/cohorts"
---

## Path Params

<ParamField path="id" type="string" required>
  The ID of the patient to add to cohorts.
</ParamField>

## Body

<ParamField body="cohortIds" type="string[]" required>
  Array of cohort IDs to add the patient to. Both cohort names and IDs are valid identifiers.
</ParamField>

## Response

<ResponseField name="cohorts" type="Cohort[]" required>
  Array of cohorts the patient was successfully added to.

  </Expandable>
</ResponseField>

## Notes

- If the patient is already in some of the specified cohorts, those will be skipped (no error thrown)
- Only successfully added cohorts are returned in the response
