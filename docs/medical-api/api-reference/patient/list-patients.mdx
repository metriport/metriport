---
title: "List Patients at Facility"
description: "Lists all Patients receiving care at the specified Facility."
api: "GET /medical/v1/patient"
---

## Query Params

<ParamField query="facilityId" type="string" Optional>
  The ID of the Facility where the patient is receiving care.
</ParamField>

## Response

All Patients receiving care at the specified Facility, or all Patients if no
Facility is specified.

<ResponseExample>

```javascript Metriport SDK
import { MetriportMedicalApi } from "@metriport/api-sdk";

const metriport = new MetriportMedicalApi("YOUR_API_KEY");

const facilityId = "018a80c4-292a-7486-a1234-9uiu76yhe234";
const patients = await metriport.listPatients(facilityId);
```
