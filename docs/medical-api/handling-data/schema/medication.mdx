The medication table contains standardized information about medications referenced across the system. This table serves as a master reference for medications and includes various coding systems for medication identification.

**Primary Keys:**
* medication_id

| Column | Data Type | Description |
| ------ | --------- | ----------- |
| medication_id | string | Unique identifier for the medication |
| status | string | Status of the medication. Possible values:<br/>– <code>active</code>: The medication is currently in use.<br/>– <code>inactive</code>: The medication is not currently in use.<br/>– <code>entered-in-error</code>: The medication record was entered in error. Refer to <a href="https://hl7.org/fhir/R4/valueset-medication-status.html" target="_blank" rel="noopener noreferrer">HL7 FHIR Medication Status</a> for more details.|
| rxnorm_code | string | RxNorm code for the medication |
| rxnorm_display | string | Human-readable description from RxNorm |
| ndc_code | string | National Drug Code for the medication |
| ndc_display | string | Human-readable description from NDC |
| data_source | string | Source of the medication data |
