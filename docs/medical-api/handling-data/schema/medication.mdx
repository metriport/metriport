The medication table contains standardized information about medications referenced across the system. This table serves as a master reference for medications and includes various coding systems for medication identification.

**Primary Keys:**
* medication_id

| Column | Data Type | Description | Terminology |
| ------ | --------- | ----------- | ----------- |
| medication_id | string | Unique identifier for the medication | - |
| rxnorm_code | string | RxNorm code for the medication | RxNorm |
| rxnorm_system | string | RxNorm coding system identifier | - |
| rxnorm_display | string | Human-readable description from RxNorm | - |
| ndc_code | string | National Drug Code for the medication | NDC |
| ndc_system | string | NDC coding system identifier | - |
| ndc_display | string | Human-readable description from NDC | - |
| atc_code | string | Anatomical Therapeutic Chemical code | ATC |
| atc_system | string | ATC coding system identifier | - |
| atc_display | string | Human-readable description from ATC | - |
| data_source | string | Source of the medication data | - |
