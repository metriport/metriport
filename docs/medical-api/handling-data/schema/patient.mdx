---
title: "Patient"
description: "Schema documentation for the Patient table in the Metriport data model"
group: "Schema"
---

The patient table contains information on patients, including demographic information such as birth date, gender, race, and ethnicity. This is the core table that contains this information.

**Normalization Note:**
This table does not include normalized coding fields as patient demographic data uses different coding systems that are not normalized in the current implementation.

**Primary Keys:**
* patient_id

| Column | Data Type | Description | Terminology |
| ------ | --------- | ----------- | ----------- |
| patient_id | string | Unique identifier for the patient | - |
| first_name | string | Patient's first name | - |
| last_name | string | Patient's last name | - |
| sex | string | Patient's sex | male, female, other, unknown |
| birth_date | date | Patient's date of birth | - |
| address | string | Patient's address | - |
| city | string | City | - |
| state | string | State or province | - |
| zip_code | string | ZIP or postal code | - |
| data_source | string | Source of the patient data | - |
