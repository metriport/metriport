---
title: "Location"
description: "Schema documentation for the Location table in the Metriport data model"
group: "Schema"
---

The location table contains information on practice and facility locations where patients receive medical care.

**Normalization Note:**
This table does not include normalized coding fields as location data uses different coding systems that are not normalized in the current implementation.

**Primary Keys:**
* location_id

**Foreign Keys:**
* managing_organization

| Column | Data Type | Description | Terminology |
| ------ | --------- | ----------- | ----------- |
| location_id | string | Unique identifier for the location | - |
| name | string | Name of the location | - |
| type | string | Type of healthcare facility | - |
| physical_type | string | Physical type of the facility | - |
| managing_organization | string | Unique identifier for the managing organization | - |
| address | string | Address of the location | - |
| address_line | string | Address line of the location | - |
| city | string | City | - |
| state | string | State or province | - |
| country | string | Country | - |
| zip_code | string | ZIP or postal code | - |
| data_source | string | Source of the location data | - |
