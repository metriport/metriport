---
title: "Condition Encounter"
description: "Schema documentation for the Condition Encounter relationship table in the Metriport data model"
group: "Schema"
---

The condition encounter table represents the relationship between conditions and encounters. This table links conditions to the specific encounters where they were diagnosed or documented.

**Normalization Note:**
This table does not include normalized coding fields as it serves as a relationship table between conditions and encounters.

**Primary Keys:**
* condition_id, encounter_id (composite key)

**Foreign Keys:**
* condition_id
* encounter_id

| Column | Data Type | Description | Terminology |
| ------ | --------- | ----------- | ----------- |
| condition_id | string | Unique identifier for the condition | - |
| encounter_id | string | Unique identifier for the encounter | - |
