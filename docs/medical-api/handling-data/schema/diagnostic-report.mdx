title: "Diagnostic Report"
description: "Schema documentation for the Diagnostic Report table in the Metriport data model"
group: "Schema"
---

The diagnostic report table contains information about diagnostic reports, including lab results, imaging studies, and other diagnostic findings.

**Primary Keys:**
* diagnostic_report_id

**Foreign Keys:**
* patient_id

| Column | Data Type | Description |
| ------ | --------- | ----------- |
| diagnostic_report_id | string | Unique identifier for the diagnostic report |
| patient_id | string | Unique identifier for the patient |
| status | string | Status of the diagnostic report. Possible values: <br/>– `registered`: The existence of the report is registered, but there is nothing yet available. <br/>– `partial`: This is a partial (e.g. preliminary) report: data in the report may be incomplete or unverified. <br/>– `preliminary`: Verified early results are available, but the report is not yet final. <br/>– `final`: The report is complete, verified, and issued. <br/>– `amended`: The report has been modified subsequent to being final, and is complete and verified. <br/>– `corrected`: Subsequent to being final, the report has been modified to correct an error in the report or referenced results. <br/>– `appended`: New information is available, but it does not change the existing report content. <br/>– `cancelled`: The report is unavailable because the measurement was not started or not completed (also sometimes called "aborted"). <br/>– `entered-in-error`: The report has been withdrawn following a previous final release (e.g. released in error). <br/>– `unknown`: The current status of the report is unknown. |
| effective_date | date | Start date of the diagnostic report |
| end_date | date | End date of the diagnostic report |
| loinc_code | string | LOINC code for the diagnostic report |
| loinc_display | string | Human-readable description of LOINC code |
| category_hl7_code | string | HL7 category code |
| category_hl7_display | string | Human-readable description of HL7 category |
| presented_form_text | string | Text content of the diagnostic report |
| data_source | string | Source of the diagnostic report data |
