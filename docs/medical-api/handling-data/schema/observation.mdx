---
title: "Observation"
description: "Schema documentation for the Observation table in the Metriport data model"
group: "Schema"
---

The observation table contains information on observations.

**Normalization Note:**
Normalization of coding systems only occurs when the source coding system is one of: LOINC or SNOMED CT. Other coding systems will not have normalized codes.

**Primary Keys:**
* observation_id

**Foreign Keys:**
* patient_id

| Column | Data Type | Description |
| ------ | --------- | ----------- |
| observation_id | string | Unique identifier for the observation |
| patient_id | string | Unique identifier for the patient |
| status | string | Status of the observation |
| effective_date | date | Start date of the observation |
| end_date | date | End date of the observation |
| loinc_code | string | LOINC code for the observation |
| loinc_display | string | Human-readable description of LOINC code |
| value | string | Value of the observation |
| units | string | Unit of measurement |
| reference_range_low | string | Lower bound of the reference range |
| reference_range_high | string | Upper bound of the reference range |
| category_hl7_code | string | HL7 category code |
| category_hl7_display | string | Human-readable description of HL7 category |
| interpretation_hl7_code | string | HL7 interpretation code |
| interpretation_hl7_display | string | Human-readable description of HL7 interpretation |
| bodysite_snomed_ct_code | string | SNOMED CT body site code |
| bodysite_snomed_ct_display | string | Human-readable description of SNOMED CT body site |
| note_text | string | Additional notes about the observation |
| data_source | string | Source of the observation data |
