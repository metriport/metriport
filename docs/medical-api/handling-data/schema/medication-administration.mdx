---
title: "Medication Administration"
description: "Schema documentation for the Medication Administration table in the Metriport data model"
group: "Schema"
---

A medication administration record represents the actual administration of a medication to a patient. This includes details about when, how, and by whom the medication was administered.

**Primary Keys:**
* medication_administration_id

**Foreign Keys:**
* patient_id
* medication_id
* practitioner_id

| Column | Data Type | Description | Terminology |
| ------ | --------- | ----------- | ----------- |
| medication_administration_id | string | Unique identifier for the medication administration | - |
| patient_id | string | Unique identifier for the patient | - |
| medication_id | string | Unique identifier for the medication | - |
| status | string | Status of the medication administration | active, completed, entered-in-error, intended, stopped, on-hold, unknown, not-taken |
| start_date | date | Start date of the medication administration | - |
| end_date | date | End date of the medication administration | - |
| dose_unit | string | Unit of measurement for the dose | - |
| dose_amount | string | Amount of medication administered | - |
| dosage_method | string | Method of administration | - |
| dosage_route | string | Route of administration | oral, intravenous, intramuscular, subcutaneous, topical |
| dosage_site | string | Anatomical site of administration | - |
| note_text | string | Additional notes about the administration | - |
| practitioner_id | string | Unique identifier for the practitioner who administered | - |
| organization_id | string | Unique identifier for the organization | - |
| data_source | string | Source of the medication administration data | - |
