---
title: "Lab Result"
description: "Schema documentation for the Lab Result table in the Metriport data model"
group: "Schema"
---

The lab result table contains information about lab test results, including the LOINC code and description, units, reference range, and result.

**Normalization Note:**
Normalization of coding systems only occurs when the source coding system is one of: LOINC or SNOMED CT. Other coding systems will not have normalized codes.

**Primary Keys:**
* lab_result_id

**Foreign Keys:**
* patient_id
* practitioner_id

| Column | Data Type | Description | Terminology |
| ------ | --------- | ----------- | ----------- |
| lab_result_id | string | Unique identifier for the lab result | - |
| patient_id | string | Unique identifier for the patient | - |
| status | string | Status of the lab result | final, preliminary, amended |
| result_date | date | Date when the result was available | - |
| collection_date | date | Date when the sample was collected | - |
| source_code_type | string | Type of coding system for source code | - |
| source_code | string | Original code from the source system | - |
| source_description | string | Human-readable description from source | - |
| normalized_code_type | string | Type of normalized coding system | LOINC, SNOMED CT |
| normalized_code | string | Standardized/normalized code | LOINC, SNOMED CT |
| normalized_description | string | Human-readable description of normalized code | LOINC, SNOMED CT |
| result | string | Value of the lab result | - |
| source_units | string | Unit of measurement from source | - |
| source_reference_range_low | string | Lower bound of the reference range from source | - |
| source_reference_range_high | string | Upper bound of the reference range from source | - |
| category_code_type | string | Type of coding system for category | - |
| category_code | string | Category code | - |
| category_description | string | Human-readable description of category | - |
| interpretation_code_type | string | Type of coding system for interpretation | - |
| interpretation_code | string | Interpretation code | - |
| interpretation_description | string | Human-readable description of interpretation | - |
| bodysite_code_type | string | Type of coding system for body site | - |
| bodysite_code | string | Body site code | - |
| bodysite_description | string | Human-readable description of body site | - |
| note | string | Additional notes about the lab result | - |
| practitioner_id | string | Unique identifier for the practitioner | - |
| data_source | string | Source of the lab result data | - |
