---
title: "Binary"
description: "A resource that represents the data of a single raw artifact as digital content accessible in its native format.  A Binary resource can contain any content, whether text, image, pdf, zip archive, etc."
---

## Properties

| Name            | Required | Type                      | Description                                                               |
| --------------- | -------- | ------------------------- | ------------------------------------------------------------------------- |
| contentType     | Yes      | code                      | MimeType of the binary content                                            |
| securityContext | No       | Reference&lt;Resource&gt; | Identifies another resource to use as proxy when enforcing access control |
| data            | No       | base64Binary              | The actual content                                                        |
| url             | No       | url                       | Uri where the data can be found                                           |

## Search Parameters

None

| implicitRules | No       | uri                                       | A set of rules under which this content was created |
