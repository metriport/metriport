---
title: "Get Nutrition Data"
description: "Returns nutrition data from all of the specified users' connected providers."
api: "GET /nutrition"
---

## Query Params

<ParamField query="userId" type="string" required>
  The Metriport user ID to create a session token for.
</ParamField>

<ParamField query="date" type="string" required>
  Day for which to pull the data. Formatted YYYY-MM-DD.

  <Info>
    For simplicity, all queries assume the UTC timezone for the specified date
    starting at 00:00 to 23:59.
  </Info>
</ParamField>

## Response

<ResponseField name="nutrition" type="Nutrition[]">
  An array of nutrition objects for each of the user's connected providers
</ResponseField>

<ResponseExample>

```javascript Metriport SDK
import { Metriport } from "@metriport/api";

const metriportClient = new MetriportDevicesApi("YOUR_API_KEY");

const response = await metriportClient.getNutritionData(
  "metriportUserId",
  "date"
);
```
