---
title: "Get Biometrics Data"
description: "Returns biometrics data from all of the specified users' connected providers."
api: "GET /biometrics"
---

## Query Params

<Snippet file="devices-api-query-params.mdx" />

## Response

<ResponseField name="biometrics" type="Biometrics[]">
  An array of biometrics objects for each of the user's connected providers

  <Info>
    For simplicity, all queries assume the UTC timezone for the specified date
    starting at 00:00 to 23:59.
  </Info>
</ResponseField>

<ResponseExample>

```javascript Metriport SDK
import { MetriportDevicesApi } from "@metriport/api-sdk";

const metriport = new MetriportDevicesApi("YOUR_API_KEY");

const response = await metriport.getBiometricsData(
  "metriportUserId", 
  "date"
);
```
