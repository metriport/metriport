    <ParamField body="hie" type="object" optional>

      document monitoring (non-ADT) / medical record refresh based on the cadence.
      <Expandable title="hie object properties" defaultOpen={false}>
          <ParamField body="enabled" type="boolean" optional>
            Enable or disable scheduled data pulls for HIE monitoring. **Defaults to enabled**.
            When disabled, no scheduled pulls will occur regardless of the frequency setting.
          </ParamField>
          
          <ParamField body="frequency" type="string" optional>
            How often scheduled data pulls should occur, can be one of `weekly` or `biweekly` or `monthly`. Defaults to `monthly`.
          </ParamField>
          
          <Info>
            See [scheduled queries](/medical-api/handling-data/scheduled-queries)

          </Info>
