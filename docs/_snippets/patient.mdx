  last name, etc.

<ParamField body="externalId" type="string">
  An external Patient ID to associate to a Patient in Metriport.
  
  <Info>
    If `externalId` is provided, it will be returned in webhook payloads and responses
    involving the Patient.
  </Info>
</ParamField>

<ParamField body="cohorts" type="string[]">
  The list of UUIDs for the cohorts this patient should be added to.

  <Note>
    See [Working With Cohorts](/medical-api/handling-data/cohorts) to learn more about how to use this field.
  </Note>
</ParamField>
