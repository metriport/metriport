<BodyField name="metadata" type="object" optional>
  The metadata parameter is an optional object that, when provided, must be a container with a 'metadata' key that holds a record of up to 
  50 custom string key-value pairs. Key names can be up to 40 characters long and values up to 500 characters long. This metadata will be 
  returned in the webhook response.You may use this to attach whatever metadata is relevant for your business use case, i.e. external ids.
</BodyField>
