# `commonwell-sdk`

SDK to simplify CommonWell API integration - by Metriport Inc.

## Usage

```
import {
  APIMode,
  CommonWell
} from "@metriport/commonwell-sdk";

const commonWell = new CommonWell(
    "CERTIFICATE",
    "PRIVATE_KEY",
    "Leopard Medical",
    "2.16.840.1.113883.3.666.7.777",
    APIMode.production
  );
```
