# `ihe-gateway-sdk`

SDK to connect to the Metriport IHE Gateway - by Metriport Inc.

## Usage

```
import {
  IHEGateway
} from "@metriport/ihe-gateway-sdk";

const iheGateway = new IHEGateway({ url });
```
